## C3 HTTP报文

#### HTTP报文结构

| 报文首部 |                                     |
| -------- | ----------------------------------- |
| 空行     | CR（回车 0x0d） + LF（换行符 0x0a） |
| 报文主体 | 应被发送的数据                      |

#### 返回结果的HTTP状态码

|      | 类别         | 原因短语           |
| ---- | ------------ | ------------------ |
| 1XX  | 信息性 | 接收的请求正在处理 |
| 2XX |  成功 | 请求正常处理完毕 |
| 3XX | 重定向 | 需进行附加操作以完成请求|
| 4XX | 客户端错误 | 服务端无法处理请求 |
| 5XX  | 服务端错误 | 服务端处理请求出错 |

##### 
| 错误码 | 说明 |
| ------- | ----------------------------------- |
| 200 OK | 请求已正常处理 |
| 204 No Content | 请求处理成功，但没有资源可返回 |
| 206 Partial Content | 表示对资源某一部分的请求 |
| 301 Move Permanently | 表示请求的资源已被分配了新的URI |
| 302 Found | 表示请求的资源已被分配了新的URI，希望用户能使用新的URI访问|
| 303 See Other | 与302类似，但明确表示客户端应当采用GET方法获取资源 |
| 304 Not Modified | 资源已找到，但未符合条件请求 |
| 400 Bad Request | 表示请求报文中存在语法错误 |
| 401 Unauthorized | 表示发送的请求需有用过HTTP认证的信息 |
| 403 Forbidden | 对请求资源的访问被服务器拒绝 |
| 404 Not Found | 服务器上没有请求的资源 |
| 500 Internal Server Error | 服务器端请求时发生错误 |
| 503 Service Unavailable | 服务器超负载或正在停机维护 |
